# 3D-Object-Part-Segmentation-and-Automatic-Semantic-Labeling-using-Vision-Language-Models

Project: 3D Object Segmentation and Automatic Part Naming System

Developed an AI-powered system for automatic segmentation of 3D objects into individual components using deep learning and computer vision techniques.
Implemented multi-view rendering pipelines to generate 2D projections from 3D meshes for improved segmentation accuracy.
Integrated state-of-the-art segmentation models to identify and isolate object parts from complex 3D geometries.
Designed an automated part-labeling framework that generates semantic names for segmented components using Large Language Models (LLMs).
Built a complete end-to-end workflow from 3D model input to segmented and labeled output visualization.
Processed and analyzed 3D mesh formats including OBJ, STL, and GLB for industrial and engineering applications.
Developed visualization tools for displaying segmented regions with corresponding part names and metadata.
Optimized inference pipelines to handle large-scale 3D models efficiently on GPU-based cloud infrastructure.
Leveraged Python, PyTorch, Open3D, Trimesh, and modern AI frameworks for model development and deployment.
Improved automation of 3D model understanding, reducing manual annotation effort in CAD and manufacturing workflows.
